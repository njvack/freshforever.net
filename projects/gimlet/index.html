Gimlet is an independently-developed software package designed to help libraries collect statistics about the usage of their public service points, as well as building a knowledge base of questions and answers. Run as a subscription-based service, Gimlet currently has more than 100 paying clients.

In this project, I have been a partner in all aspects of software design, customer support, billing, and system administration.

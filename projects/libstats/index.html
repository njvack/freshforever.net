Project page

Libstats allows a library reference desk to build a searchable knowledge base of reference questions and collect statistics about interactions with customers. Designed to replace paper forms, Libstats was built from the ground-up to minimize the amount of staff time needed for question entry. I was the initial and lead designer and developer on this project.

Libstats was released as an open-source project in 2005. To date, it has been downloaded more than 3,000 times.
